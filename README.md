# HR Competence Management System

A web-based HR application designed to support decision-making for HR professionals and Team Leaders.

## Key Features
- Employee competence management
- Skill gap and shortage analysis
- Demand vs supply visualization
- Role-based access (HR, Team Leader, Project Manager)
- CSV export for reporting

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Azure DevOps (CI)
- GitHub (Portfolio)

## Purpose
This project was developed as part of a practical training (LIA) and academic project, focusing on digital competence management and decision support in organizations.
